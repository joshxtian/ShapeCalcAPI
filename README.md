## Usage

### Perimeter of Rectangle

To calculate perimeter of rectangle:

Route: `api/area-square`
Body: 

  `req: {
    "width": <int>,
    "length": <int>
  }`


### Volume of cuboid

To calculate Volume of Cuboid:

Route: `api/vol-cub`
Body: 

  `req: {
    "length": <int>,
    "width": <int>,
    "height": <int>
  }`



### Area of Square

To calculate Area of Square:

Route: `api/area-square`
Body: 

  `req: {
    "side": <int>
  }`
